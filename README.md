# Gamified Habit Tracker

A full-stack habit tracking application with RPG-style progression, habit completions, streaks, XP, and character attributes.

The application is being built in deliberate phases, beginning with the core habit-tracking loop. Milestones and achievements are planned as post-MVP features.

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### Backend

- C#
- ASP.NET Core Web API
- Entity Framework Core

### Database

- PostgreSQL

### Testing

- xUnit
